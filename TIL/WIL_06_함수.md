함수 입력은 **매개변수** 모든 타입을 취할 수 있고 모든 타입을 반환 가능하다.

-정의하기(define) : 0개 또는 1개 이상의 매개변수를 갖는다.
-호출하기(call) : 0개 또는 1개 이상의 결과를 얻는다.

def whatis(thing):
    if thing is None:
        print(thing, "is None")
    elif thing:
        print(thing, "is True")
    else:
        print(thing, "is False")

whatis(None)
>>> None is None
